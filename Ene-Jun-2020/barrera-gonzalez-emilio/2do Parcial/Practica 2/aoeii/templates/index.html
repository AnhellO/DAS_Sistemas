    ABOUT THE PROJECT

    A pretty basic and open source implementation of HEROKUAPP's API for the better game of the trilogy, Age Of Empires II.
    It includes basic info about the different Civilizations, Units and Structures of the game. basically, you have two options for each category: You can individually search for them or you can take a dive into each of the lists

    FEEL FREE TO CONTRIBUTE

    As stated before, this is an OPEN SOURCE IMPLEMENTATION of an already existing API
    so, if you want to help, you can do so by forking the repo over on GITHUB
